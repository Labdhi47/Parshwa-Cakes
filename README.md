# Parshwa-Cakes

A simple and responsive static website built using HTML, CSS, and JavaScript.
This project is lightweight, fast, and easy to deploy on any static hosting platform.

## Features

✅ Fully responsive design

✅ Clean and modern UI

✅ Image gallery with thumbnails

✅ Modal popup for full image view

✅ Organized folder structure

✅ Easy to customize

## Technologies Used

HTML5 – Structure

CSS3 – Styling & Layout (Flexbox/Grid)

JavaScript (Vanilla JS) – Interactivity

## Getting Started
1️⃣ Clone the Repository

```bash
git clone https://github.com/Labdhi47/Parshwa-Cakes.git
```

2️⃣ Open the Project
Simply open index.html in your browser.

OR

Use Live Server (Recommended if using VS Code).
